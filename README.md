# POKÉdado 🎲⚡

**POKÉdado** é um projeto desenvolvido para auxiliar mestres de RPG em campanhas que utilizem sistemas baseados no universo de Pokémon. Com ferramentas intuitivas e recursos personalizados, o site facilita a criação e o gerenciamento de aventuras, permitindo que os mestres se concentrem na narrativa e na diversão dos jogadores.
